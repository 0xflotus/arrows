Graph Diagram Example

This page demonstrates including a simple graph diagram in an HTML page. View the source to see how it works.

The important parts are:

  • A figure tag, to contain the diagram.
  • Inside the figure tag, HTML markup describing the graph.
  • CSS to hide the markup.
  • A tiny bit of javscript that calls the rendering library.

With all this in place, the figure will end up containing two children; firstly the markup tag that's in the source, and secondly an SVG tag generated by the library.
